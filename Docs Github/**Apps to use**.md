#Apps to use

## DrawBot:
a free macOS app where you write simple Python scripts to generate 2D graphics. In type design, people often use it to make proofs, specimens, visual tests, PDFs, images, and sometimes animations.

# MacDown: 
an open-source Markdown editor for macOS. This is usually for writing notes, documentation, process logs, or README-style text files in a clean plain-text format with preview.

# Fontra Pak: 
a self-contained version of Fontra, which is a browser-based font editor. Its purpose is to let you view and edit fonts, especially in a variable-font-focused workflow, using your default browser.
